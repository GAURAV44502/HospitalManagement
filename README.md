I developed a Hospital Management System using Spring Boot and Hibernate to practice building backend applications with RESTful APIs.
I created multiple endpoints to perform CRUD (Create, Read, Update, Delete) operations and tested all APIs using Postman.<br>
⚙️ Tech Stack:<br>
Backend: Java, Spring Boot<br>
ORM: Hibernate, JPA<br>
Database: MySQL<br>
Tools: Postman (for API testing), IntelliJ IDEA<br>
Build Tool: Maven<br>
